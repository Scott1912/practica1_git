Desarrollo Web

¿Qué es la web?
World Wide Web, o simplemente Web, es el universo de información accesible a través de Internet, una fuente inagotable del conocimiento humano.
Tecnologias de la web
Existen numerosas tecnologías web, algo que si bien es una ventaja en el diseño y desarrollo de sitio web, puede llegar a complicar la elección o la toma de decisión de cuál utilizar y para qué. Por eso es imprescindible explicar en detalle el proyecto que se quiere desarrollar, las funciones que necesitará el desarrollo web, y en base a eso, dejarse asesorar por los desarrolladores a la hora de determinar qué tecnologías web se usarán.
Algunas tecnologías fronted o del lado del cliente son:

  • JavaScript
  • HTML
  • CSS
Por otro lado, las tecnologías backend o del lado del servidor son:
  • PHP
  • Python
  • MySQL
HTML
Este es un lenguaje de programación usado para la creación de sitios web. Como las tecnologías evolucionan a pasos agigantados, HTML está en constante renovación, la versión actual es HTML5. Consiste en un conjunto de códigos cortos, que se clasifican como archivos de textos en las etiquetas. Es decir, el texto se guarda en un archivo llamado HTML que se puede encontrar a través de los buscadores. Cada una de las etiquetas generadas tiene diferentes funciones. En resumen, HTML sirve para describir el contenido de un sitio web, como la información estructurada de párrafos, imágenes, etc. Por eso, HTML es una de las tecnologías web indispensables para cualquier desarrollador.
CSS
Es una de las tecnologías web a la que más recurren los programadores gráficos a la hora de desarrollar sus proyectos. Es un lenguaje de diseño gráfico que sirve, entre otras muchas funciones, para indicar la representación visual de las estructuras HTML, así como las estructuras HTML definidas anteriormente deben representarse visualmente. De esta manera, el lenguaje CSS sirve para acotar y trazar el aspecto visual de las etiquetas generadas por el HTML.
JavaScript
Al usar este lenguaje de programación multiplataforma es posible dar una mayor interactividad y dinamismo a los sitios web. Y esto es así porque permite crear tanto animaciones como objetos, localizar errores en formularios, cambiar elementos web de manera intuitiva y crear cookies, etc. Además, también permite desarrollar aplicaciones tan potentes como Facebook o Twitter. Por esto, JavaScript es una de las tecnologías web más usadas en la actualidad.

Etiquetas de texto

Las etiquetas de texto nos ayudaran a escribir la estructura de texto de nuestra página web algunas de estas son:

Parrafo

Esta estiqueta agregara los parrafos a nuestra página sin un formato especifico, es decir, si en nuestro editor le queremos dar una estructura especifica, en la web no aparecera como en el editor, para hacer esto usaremos la etiqueta pre
Ejemplo:

< p >
Así es como se escribirá en nuestro editor de código
< /p >

Parrafo con preformato

Con esta etiqueta el formato que tenga nuestro parrafo en el editor de código, será el mismo que aparecera en nuestra web
Ejemplo de como aparece en el editor de código:

< pre > 

            En nuestro editor de código aparecerá de esta forma en que lo estamos escribiendo 
                     y en la web aparecerá de igual forma 
            incluido 
                 cada 
            salto de línea que hagamos dentro del editor, cada espacio, etc.
< /pre >

Como aparece en la web: 

        En nuestro editor de código aparecerá de esta forma en que lo estamos escribiendo 
                 y en la web aparecerá de igual forma 
        incluido 
             cada 
        salto de línea que hagamos dentro del editor, cada espacio, etc.

Listas

Nos permiten escribir listas ya sean ordenadas o desordenadas. El formato de la etiqueta es el siguiente: 

        Lista desordenada:
        <ul>
        <li>Elemento de mi lista</li>
        <li>Elemento d emi lista</li>
        </ul>
    
        Lista ordenada:
        <ol>
        <li>Elemento de mi lista</li>
        <li>Elemento d emi lista</li>
        </ol>

Tablas

Una tabla es un conjunto estructurado de datos distribuidos en filas y columnas.
Una tabla básica puede ser declarada usando tres elementos, a saber, table (el contenedor principal), tr (representando a las filas ) y td (representando a las columnas).
